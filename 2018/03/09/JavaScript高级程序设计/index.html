typeof

ECMAScript有5种基本类型数据：

  • undefined
  • null
  • boolean
  • number
  • string

另外还有一种复杂的数据类型

  • object

typeof就是用来检测变量的数据类型的，typeof可能会返回以下值：

  • undefined
  • boolean
  • string
  • number
  • object
  • function
